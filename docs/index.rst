Willkommen beim Pydrofoil Random Testfallgenerator!
===================================================

.. toctree::
   :maxdepth: 2
   :caption: Contents:

   modules

* :ref:`genindex`

Dieses Projekt ist ein automatisierter Random-Testfallgenerator für den
**RISC-V Emulator von Pydrofoil**, basierend auf der **Pydrofoil Scripting API**.

Der Generator erstellt Registerinhalten an hand ihrer Typen. Womit Tests kombiniert werden um verschiedene Zustände abzudecken.

Ziel ist es, durch vielfältige Randomisierung eine robuste Testabdeckung und
Fehlersuche im Emulator sicherzustellen, durch die Integration von
**Property-Based Testing mit Hypothesis** möglich.

Features
--------

- ✅ Vollständige Randomisierung von Registern
- ✅ Erweiterbar für bestimmte Teststrategien oder Architekturerweiterungen
